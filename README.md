# React Todo List (Mantine + Redux + i18n)

Современное приложение для управления задачами, построенное на стеке React + TypeScript с использованием продвинутых библиотек для UI и управления состоянием.

[Результат](https://to-do-app-blond-gamma.vercel.app/) 

## ✨ Особенности

- **Управление задачами**: Добавление, удаление и переключение статуса выполнения.
- **Фильтрация**: Просмотр всех, активных или завершенных задач.
- **Redux State Management**: Использование Redux Toolkit для предсказуемого состояния.
- **Многоязычность**: Поддержка трех языков (Белорусский, Русский, Английский) через `i18next`.
- **Темная тема**: Встроенная поддержка светлой и темной тем оформления.
- **Сохранение данных**: Задачи автоматически сохраняются в `localStorage` и доступны после перезагрузки.
- **Адаптивный дизайн**: Полная отзывчивость благодаря Mantine UI.

## 🛠 Технологии

- **React 18** + **TypeScript**
- **Vite** (сборщик проекта)
- **Mantine UI v7** (компоненты и стилизация)
- **Redux Toolkit** (управление состоянием)
- **i18next** (интернационализация)
- **Heroicons** (иконки)

## 🚀 Как запустить

1. Убедитесь, что у вас установлен [Node.js](https://nodejs.org/)
2. Клонируйте репозиторий с помощью команды git clone
3. Установите зависимости с помощью команды npm install
4. Запустате проект в режиме разработки с помощью команды npm run dev
